Model 1: Gradient Boosted Trees

Fitting a boosted regression model

  • Use gaussian for a continuous response.
  • interaction.depth controls how many splits in each tree
  • shrinkage is the learning rate.
    • If small, may need more trees to converge
  • the way the categorical variables are split is that it will randomly choose a subset, and then some hosptials will go left and some will go right.
set.seed(123)

boost.costs = gbm(log_cost ~ APR.MDC.Description + Facility.Name + Age.Group + Gender + APR.Severity.of.Illness.Code + Payment.Typology.1 + Type.of.Admission + Patient.Disposition,
    data = train_data,
    distribution = "gaussian",
    n.trees = 10000,
    interaction.depth = 4,
    shrinkage = 0.01,
    cv.folds = 5,
    verbose = FALSE)
